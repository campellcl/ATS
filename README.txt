The data provided by the Appalachian Trail Conservancy is in Shapefile form: https://en.wikipedia.org/wiki/Shapefile. 
The Python package capable of parsing Shapefiles is pyshp, available here: https://pypi.python.org/pypi/pyshp
Use Selenium to parse trail journal data from: http://www.trailjournals.com/journals/appalachian_trail/
Selenium and Xscrape tutorial available: http://student.cs.appstate.edu/~rmp/scrape/
